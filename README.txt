# MonIAna Web

Archivos incluidos:
- index.html
- styles.css
- script.js
- images/logo-moniana.png
- images/hero-moniana.jpg

Para probar en Vercel:
1. Sube estos archivos a un repositorio de GitHub.
2. Entra en Vercel.
3. Importa el repositorio.
4. Publica el proyecto.

Para Hostinger:
Puedes subir estos archivos desde el Administrador de archivos al directorio public_html.
